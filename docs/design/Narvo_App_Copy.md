## 7. Formatting Guidelines
- **Brackets `[]`**: Used for technical/instrument commands or status indicators (e.g., `[Start Broadcast]`).
- **Tabular Numerals**: Use monospaced numbers for all data values (speeds, times, storage).
- **All Caps (Selective)**: Use sparingly for Top-level category headers only (e.g., LATEST).
