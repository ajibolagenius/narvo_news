# Narvo Overview

Narvo is a precision-engineered news broadcast platform that transforms fragmented information into high-fidelity, audio-enhanced narrative stories. Built for global audiences with a primary focus on African markets, it leverages advanced AI to deliver trustworthy broadcasts in authentic local voices.

## Mission
To democratize access to high-quality, trustworthy news by stripping away bloat and delivering content through an audio-first, minimalist experience.

## Core Pillars
- **Strict Minimalism**: The "Swiss Grid" design language with a 10% primary color rule.
- **Audio-First**: A broadcast-style delivery model optimized for eyes-busy and low-bandwidth environments.
- **Trusted Authority**: Integrated fact-checking and transparent paraphrasing logic.
- **Unified Engineering**: A robust Monorepo for seamless Mobile and Web consistency.

## Shift from Version 1
- **Design**: Transitioned from "Bento" to a rigid, technical Swiss Matrix.
- **Content**: Moved from simple summarization to narrative "Broadcast Stories."
- **Technology**: Adopted a Monorepo with Google Gemini, Vector DBs (RAG), and premium TTS (ElevenLabs).
