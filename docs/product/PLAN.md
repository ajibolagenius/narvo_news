* Setup Next.js PWA scaffold with Tailwind CSS.
* Configure Supabase for Auth and PostgreSQL.

## Phase 3: UI & UX (Weeks 6-8)

* Develop the Bento Grid Home Feed.
* Build the Onboarding "Interest Picker".
* Implement the "Litverse-inspired" persistent audio player.

## Phase 4: Offline & Optimization (Weeks 9-10)

* Configure IndexedDB for offline audio storage.
* Optimize for 3G connection speeds (LCP < 2.5s).
